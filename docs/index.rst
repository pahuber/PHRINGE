SYGN
====

SYGN is a **SY**\ nthetic photometry data **G**\ enerator for **N**\ ulling interferometers. It can generate realistic photometry data as a function of wavelength and time for space-based nulling interferometers. It can model a variety of different interferometer architectures and account for astrophysical and instrumental noise sources.
